Title: Real-Time Traffic Speed Prediction for Urban Road Networks with an Interactive Web Interface

Description:
This project predicts traffic speed in urban road networks using machine learning techniques. It also includes an interactive web interface to visualize and input data for predictions.

Features:
- Traffic speed prediction using ML model
- User-friendly web interface
- Data preprocessing and analysis
- Real-time input and prediction

Technologies Used:
- Python
- Flask (for web app)
- Pandas, NumPy
- Scikit-learn
- HTML, CSS

Project Structure:
- app.py → Main application file
- templates/ → HTML pages
- static/ → CSS, JS files
- model/ or data/ → Dataset or trained model

How to Run:
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000/

Output:
The system predicts traffic speed based on input parameters and displays results through a web interface. The system provides future traffic for next 30 minutes with 5 minutes interval.
